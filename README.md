# 📊 DataLyze — Intelligent Data Analysis Platform

> Upload a dataset. Explore it visually. Share findings as a dashboard. No code required.

DataLyze is an open-source, lightweight, offline-first, mobile & browser based data analysis tool. Upload a CSV or Excel file, generate interactive charts and quality reports, and save your findings as a shareable dashboard — without writing any code.

## Quick Start

```bash
git clone https://github.com/VidalNat/datalyze.git
cd datalyze
pip install -r requirements.txt
streamlit run app.py
```

Open `http://localhost:8501` in your browser. That's it.
